Docker Desktop

代替: Podman, Rancher Desktop


Elasticsearch → Elastic License

検索エンジンはオープン → 一部機能有料化
代替: OpenSearch (AWS主導)


Terraform → HashiCorp License

代替: OpenTofu (Linux Foundation)


MongoDB → SSPL License

クラウド提供を制限
代替: FerretDB, DocumentDB
